Welcome to Project Hopper!
Use /bin/bash to launch bash
Use ./startsql.sh <SQLFILE> <DATABASENAME> to import your SQL database
Use ./savesql.sh <FILENAME.SQL> <DATABASE> to export your SQL database
After exporting, remember to push your changes using Git!
Use 'cat /etc/motd' to see this message again.
